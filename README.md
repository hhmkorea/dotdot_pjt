# dotdot_pjt

### *** 프로젝트 정보 *** 
* 기본 패키지 명 : dotdot 
* 프로젝트명 : board 

### *** 개발환경 *** 
* 개발 언어 : JAVA
* 개발 도구 : IntelliJ
* 프레임워크 : Spring Boot
* WAS : Spring Boot에 Tomcat Server 내장됨.
* DB : mariaDB
* Builder : Gradle
* UX : Thymeleaf, jQuery(Ajax 비동기 처리)
* UI : Bootstrap
* CI & CD : Docker 

### *** 폴더별 설명 *** ###
* board : 게시판
